# dogie

## 项目介绍
一个集成 **Cartographer 建图**、**三维重定位** 以及 **Move Base 2D 导航** 的通用代码库。

## 环境需求
- **操作系统**: Ubuntu 20.04
- **ROS 发行版**: Noetic
- **Python 版本**: 3.8

## 使用须知
需要具备以下模块的编译能力，即可驾驭此代码库：
1. 能够编译 **Cartographer** 建图功能包。
2. 能够编译 **FAST-LIO** 系列算法。
3. 能够编译 **Move Base** 导航功能包。
